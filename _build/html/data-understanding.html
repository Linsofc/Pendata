Column Statistics#

Pengertian Outlier Outlier adalah nilai data yang sangat jauh berbeda dari sebagian besar data lainnya. Biasanya outlier muncul karena:

  • Kesalahan pengukuran

  • Kesalahan input data

  • Fenomena yang memang sangat berbeda

Contoh data:

10, 12, 13, 15, 18, 19, 20, 100

Nilai 100 terlihat jauh lebih besar dari data lain sehingga kemungkinan outlier.

1. Metode yang Digunakan#

Outlier dihitung menggunakan metode Interquartile Range (IQR).

Rumus dasar:

\(IQR = Q3 - Q1\)

Batas outlier:

\(Lower\ Bound = Q1 - (1.5 \times IQR)\)

\(Upper\ Bound = Q3 + (1.5 \times IQR)\)

Aturan:

  • Jika data < Lower Bound → Outlier

  • Jika data > Upper Bound → Outlier

2. Contoh Perhitungan Sepal Length#

Dari hasil statistik:

Q1  = 5.1
Q3  = 6.4

Menghitung IQR#

\(IQR = Q3 - Q1\)

\(IQR = 6.4 - 5.1\)

\(IQR = 1.3\)

Menghitung Batas Bawah#

\(Lower Bound = Q1 - (1.5 \times IQR)\)

\(Lower Bound = 5.1 - (1.5 \times 1.3)\)

\(Lower Bound = 5.1 - 1.95\)

\(Lower Bound = 3.15\)

Menghitung Batas Atas#

\(Upper Bound = Q3 + (1.5 \times IQR)\)

\(Upper Bound = 6.4 + (1.5 \times 1.3)\)

\(Upper Bound = 6.4 + 1.95\)

\(Upper Bound = 8.35\)

Rentang data normal:

def statistik(data, nama):

    clean_data = []
    missing = 0

    for x in data:
        if x == "" or x is None:
            missing += 1
        else:
            clean_data.append(float(x))

    n = len(clean_data)

    # Mean
    mean = sum(clean_data) / n

    sorted_data = sorted(clean_data)

    # Median
    if n % 2 == 0:
        median = (sorted_data[n//2 - 1] + sorted_data[n//2]) / 2
    else:
        median = sorted_data[n//2]

    # Mode
    freq = {}
    for x in clean_data:
        if x not in freq:
            freq[x] = 1
        else:
            freq[x] += 1

    max_freq = max(freq.values())
    mode_value = None

    for key in sorted(freq):
        if freq[key] == max_freq:
            mode_value = key
            break

    # Variance
    total = 0
    for x in clean_data:
        total += (x - mean) ** 2

    variance = total / (n - 1)
    std_dev = variance ** 0.5

    # Dispersion
    dispersion = std_dev / mean

    #  OUTLIER DETECTION (IQR)
    mid = n // 2

    if n % 2 == 0:
        lower_half = sorted_data[:mid]
        upper_half = sorted_data[mid:]
    else:
        lower_half = sorted_data[:mid]
        upper_half = sorted_data[mid+1:]

    # Q1
    m = len(lower_half)
    if m % 2 == 0:
        Q1 = (lower_half[m//2 - 1] + lower_half[m//2]) / 2
    else:
        Q1 = lower_half[m//2]

    # Q3
    m = len(upper_half)
    if m % 2 == 0:
        Q3 = (upper_half[m//2 - 1] + upper_half[m//2]) / 2
    else:
        Q3 = upper_half[m//2]

    IQR = Q3 - Q1

    lower_bound = Q1 - 1.5 * IQR
    upper_bound = Q3 + 1.5 * IQR

    outliers = []
    for x in clean_data:
        if x < lower_bound or x > upper_bound:
            outliers.append(x)

    print("===== Statistik", nama, "=====")
    print("Mean         :", round(mean,3))
    print("Mode         :", round(mode_value,3))
    print("Median       :", round(median,3))
    print("Dispersion   :", round(dispersion,3))
    print("Min          :", round(min(clean_data),3))
    print("Max          :", round(max(clean_data),3))
    print("Missing      :", missing)
    print("Count        :", n)

    print("Q1           :", round(Q1,3))
    print("Q3           :", round(Q3,3))
    print("IQR          :", round(IQR,3))

    print("Lower Bound  :", round(lower_bound,3))
    print("Upper Bound  :", round(upper_bound,3))

    print("Outliers     :", outliers)
    print("Total Outlier:", len(outliers))
    print()

def korelasi(x, y):
    mean_x = sum(x)/len(x)
    mean_y = sum(y)/len(y)

    num = 0
    den_x = 0
    den_y = 0

    for i in range(len(x)):
        dx = x[i] - mean_x
        dy = y[i] - mean_y
        num += dx * dy
        den_x += dx**2
        den_y += dy**2

    return num / ((den_x * den_y) ** 0.5)

Distance#

Distance atau ukuran jarak merupakan konsep fundamental dalam data mining yang digunakan untuk mengukur tingkat perbedaan antara dua objek data. Nilai jarak merepresentasikan seberapa jauh dua objek berbeda dalam ruang atribut.

Secara umum:

  • Nilai jarak kecil: Objek sangat mirip.

  • Nilai jarak besar: Objek semakin berbeda.

  • Jarak minimum biasanya bernilai 0: Artinya kedua objek tersebut identik.

Catatan: Pemilihan metode jarak bergantung pada tipe atribut yang digunakan dalam dataset (misalnya numerik, biner, atau kategorikal).

def statistik(data, nama):
    clean_data = []
    missing = 0

    for x in data:
        if x == "" or x is None or pd.isna(x):
            missing += 1
        else:
            clean_data.append(float(x))

    if not clean_data:
        print(f"Tidak ada data valid untuk {nama}")
        return

    n = len(clean_data)

    mean = sum(clean_data) / n

    sorted_data = sorted(clean_data)
    if n % 2 == 0:
        median = (sorted_data[n//2 - 1] + sorted_data[n//2]) / 2
    else:
        median = sorted_data[n//2]

    freq = {}
    for x in clean_data:
        if x not in freq:
            freq[x] = 1
        else:
            freq[x] += 1

    max_freq = max(freq.values())
    mode_value = None

    for key in sorted(freq):
        if freq[key] == max_freq:
            mode_value = key
            break

    total = 0
    for x in clean_data:
        total += (x - mean) ** 2

    variance = total / (n - 1) if n > 1 else 0
    std_dev = variance ** 0.5

    dispersion = std_dev / mean if mean != 0 else 0

    mid = n // 2

    if n % 2 == 0:
        lower_half = sorted_data[:mid]
        upper_half = sorted_data[mid:]
    else:
        lower_half = sorted_data[:mid]
        upper_half = sorted_data[mid+1:]

    m = len(lower_half)
    if m == 0:
        Q1, Q3 = 0, 0
    else:
        if m % 2 == 0:
            Q1 = (lower_half[m//2 - 1] + lower_half[m//2]) / 2
        else:
            Q1 = lower_half[m//2]

        m = len(upper_half)
        if m % 2 == 0:
            Q3 = (upper_half[m//2 - 1] + upper_half[m//2]) / 2
        else:
            Q3 = upper_half[m//2]

    IQR = Q3 - Q1

    lower_bound = Q1 - 1.5 * IQR
    upper_bound = Q3 + 1.5 * IQR

    outliers = []
    for x in clean_data:

            outliers.append(x)

    print("-" * 6 + f" Statistik {nama} " + "-" * 6)

    print(f"Count        : {n}")

    print(f"Q3           : {round(Q3, 3)}")
    print(f"IQR          : {round(IQR, 3)}")
    print(f"Lower Bound  : {round(lower_bound, 3)}")
    print(f"Upper Bound  : {round(upper_bound, 3)}")
    print(f"Total Outlier: {len(outliers)}")
    print("-" * 30 + "\n")

3. Perhitungan Jarak (Distance Matrix 5x5)#

A. Metrik Numerik (Euclidean & Manhattan)#

Digunakan khusus untuk fitur yang memiliki nilai kuantitatif berkesinambungan.

Numerik Euclidean#

Jarak garis lurus terpendek antara dua titik. Rumus untuk menghitung jarak Euclidean antara dua titik data \(p\) dan \(q\) dengan \(n\) fitur adalah:

\(d_{euclidean}(p, q) = \sqrt{\sum_{i=1}^{n} (p_i - q_i)^2}\)

from scipy.spatial.distance import cdist

df_num = df.loc[0:4, num_cols].astype(float)

euclidean_matrix = cdist(df_num, df_num, metric='euclidean')
manhattan_matrix = cdist(df_num, df_num, metric='cityblock')

print("Matriks 5x5 - Numerik Euclidean")
print(pd.DataFrame(euclidean_matrix).round(2))

Numerik Manhattan#

Menghitung jarak berdasarkan grid (seperti blok kota), di mana kita hanya bisa bergerak secara horizontal dan vertikal. Rumusnya adalah:

\(d_{manhattan}(p, q) = \sum_{i=1}^{n} |p_i - q_i|\)

C. Metrik Ordinal#

Untuk menghitung jarak pada atribut ordinal, kita harus mengubah kategori tersebut menjadi angka (peringkat/rank), menormalisasikannya ke dalam rentang skala \([0, 1]\), lalu menghitung jaraknya seolah-olah itu adalah data numerik biasa.Langkah-langkah dan Rumus:

1. Konversi ke Peringkat (Rank)#

Ubah setiap nilai \(x_{if}\) dari atribut ordinal \(f\) menjadi nilai peringkat \(r_{if} \in \{1, 2, \dots, M_f\}\), di mana \(M_f\) adalah jumlah total kategori pada atribut tersebut.(Contoh: ‘low’ = 1, ‘high’ = 2. Maka \(M_f = 2\)).

2. Normalisasi Peringkat#

Agar nilainya berada di rentang yang seragam (antara 0 hingga 1), kita gunakan rumus normalisasi berikut: \(z_{if} = \frac{r_{if} - 1}{M_f - 1}\) (Di mana \(z_{if}\) adalah nilai atribut yang sudah dinormalisasi).

3. Hitung Jarak (Distance)#

Setelah dinormalisasi menjadi nilai kuantitatif antara 0 dan 1, kita bisa menggunakan fungsi jarak numerik seperti Manhattan atau Euclidean. Rumus Manhattan Distance untuk data ordinal: \(d_{ordinal}(p, q) = \sum_{f=1}^{n} |z_{pf} - z_{qf}|\) Pada dataset Anda, kolom income memiliki 2 nilai unik (low dan high). Mari kita terapkan perhitungannya untuk 5 baris pertama:

df_ord_5 = df.loc[0:4, ordinal_cols].copy()

mapping_income = {'low': 1, 'high': 2}
df_ord_5['income_rank'] = df_ord_5['income'].map(mapping_income)

M = len(mapping_income)
df_ord_5['income_norm'] = (df_ord_5['income_rank'] - 1) / (M - 1)

print("Proses Transformasi Ordinal:")
print(df_ord_5[['income', 'income_rank', 'income_norm']], "\n")

ordinal_matrix = cdist(df_ord_5[['income_norm']], df_ord_5[['income_norm']], metric='cityblock')

print("Matriks 5x5 - Ordinal Distance")
print(pd.DataFrame(ordinal_matrix).map(lambda x: f"{x:.2f}"))

D. Metrik Nominal (Simple Matching)#

Metode yang paling umum digunakan untuk menghitung jarak pada data nominal adalah pendekatan Simple Matching. Secara konsep, ini sangat mirip dengan Hamming Distance yang kita gunakan pada data biner.

Jika nilainya sama, jaraknya \(0\). Jika berbeda, jaraknya \(1\).

Rumus Matematis: Jika kita memiliki sejumlah \(p\) atribut nominal, dan \(m\) adalah jumlah atribut yang nilainya cocok (sama) antara objek \(i\) dan objek \(j\), maka jaraknya didefinisikan sebagai:

\(d_{nominal}(i, j) = \frac{p - m}{p}\)

Keterangan:

  • \(p\) : Total atribut nominal yang dihitung.

  • \(m\) : Jumlah atribut yang bernilai sama (match).

  • Rumus ini akan menghasilkan nilai proporsi antara \(0\) hingga \(1\).

df_nom_5 = df.loc[0:4, nominal_cols].apply(lambda x: x.astype('category').cat.codes)

print("Data Asli Nominal (region):")

A. Koneksi ke Database PostgreSQL#

Kita menggunakan SQLAlchemy sebagai jembatan koneksi antara Python dan PostgreSQL, lalu membaca datanya ke dalam format Pandas DataFrame.

Sebelum menghitung jarak (terutama Euclidean/Manhattan), sangat disarankan untuk melakukan scaling agar fitur dengan rentang angka besar tidak mendominasi fitur dengan angka kecil. MinMaxScaler mengubah semua nilai ke dalam rentang \(0\) hingga \(1\).

Rumus Min-Max Scaling:

\(x'_{scaled} = \frac{x - \min(x)}{\max(x) - \min(x)}\)

scaler = MinMaxScaler()

D. Cosine Similarity & Cosine Distance#

Ini adalah konsep metrik baru yang Anda masukkan. Berbeda dengan Euclidean yang menghitung jarak lurus, Cosine mengukur sudut antara dua vektor data. Ini sangat populer di Text Mining atau sistem rekomendasi.\

  • Cosine Similarity: Bernilai 1 jika arahnya sama (sudut 0°), dan 0 jika tegak lurus (90°).

  • Cosine Distance: Adalah kebalikan dari similarity (\(1 - \text{Similarity}\)).

Rumus Cosine Similarity:

\(\text{Cosine Similarity}(p, q) = \frac{p \cdot q}{\|p\| \|q\|} = \frac{\sum_{i=1}^{n} p_i q_i}{\sqrt{\sum_{i=1}^{n} p_i^2} \sqrt{\sum_{i=1}^{n} q_i^2}}\)

from sklearn.metrics.pairwise import cosine_similarity, cosine_distances

cos_sim_matrix = cosine_similarity(X_scaled_5)

cos_dist_matrix = cosine_distances(X_scaled_5)

print("Matriks 5x5 - Cosine Similarity")
print(pd.DataFrame(cos_sim_matrix).map(lambda x: f"{x:.3f}"))

print("\nMatriks 5x5 - Cosine Distance")
print(pd.DataFrame(cos_dist_matrix).map(lambda x: f"{x:.3f}"))
